# Student Management System

A modern web application for tracking and managing student performance.

## Features

- Secure login with password validation
- Add students with scores
- Search and filter students
- Color-coded performance badges
- Remove students
- Responsive design

## Tech Stack

- React + Vite
- CSS3 (Custom styling)

## Installation

```bash
npm install
npm run dev
```

## Login Requirements

- Username: Any username
- Password: Min 6 characters, 1 uppercase, 1 special character
